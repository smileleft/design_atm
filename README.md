# Design Atm Project #

## How to Clone
```
git clone github.com/smileleft/design_atm.git
```

## Move to Project Root
```
cd design_atm
```

## Install Dependency
```
pip install -r requirements.txt
# pytest would be installed
```

## Run 
```
python main.py
```

## Run Tests
```
# on the project root directory
pytest -v
```
